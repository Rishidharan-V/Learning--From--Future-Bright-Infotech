1.Abstract

The primary objective of this project is to leverage CNNs for analysising and identifying the condition of Agricultural produce,specifically wheat,rice,fruits and vegetables By caputring three still images(font,side,back)using a webcam, key body landmark are detected, and distance between these points are calculated for precies measurement

2.Introduction

This project presents a CNN-based AI system for automated detection of plant diseases in rice and wheat, and for evaluating the freshness and shelf life of fruits and vegetables through image analysis, enhancing agricultural efficiency and sustainability.

3.Project Features

  • Real-time body measurement analysis
  • Accurate landmark detection
  • Compatibility with webcams

Key technologies Used

  1. HRNet for high-resolution keypoint detection
  2. Python fpr AI model implementation
  3. OpenCV for image processing
