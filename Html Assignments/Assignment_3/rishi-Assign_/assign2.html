1.Abstract

This project aim to develop an accurate nody measurement system using a hybrid approch that combines Phase 3d and HRNet. By caputring three still images(font,side,back)using a webcam, key body landmark are detected, and distance between these points are calculated for precies measurement

2.Introduction

The purpose of this project is to develop a highly accurate body measurement system that leverage state-of-the-art technologies refinement, enabling real-time Analysis of human body dimensions.

3.Project Features

  • Real-time body measurement analysis
  • Hybrid AI approch using Phase 3D & HRNet
  • Accurate landmark detection
  • Compatibility with webcams

Key technologies Used

  1. Phase 3D for 3D Modeling
  2. HRNet for high-resolution keypoint detection
  3. Python fpr AI model implementation
  4. OpenCV for image processing
