1: Abstract

This Project aims to develop an accurate measurement system using a hybrid approch that combaines PHASE 3D AND HRNET. By capturing three main still images (front,side,back) with a Webcam, key fruit landmark are detected and distances between points are calculated.

2:Introduction

The pupose of this project is to develop a highly accurate fruit measurementsystem that leverage state-of-the-art technology for keypoint detection and deep learning refinement
